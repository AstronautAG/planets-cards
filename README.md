# Flashcards: The Objects in our Solar System

## What is it?
This is a cool, interactive, and fun space flashcards website for anyone who wants to learn a few facts about the objects in our solar system.

## Inspiration
Space facts can be very unusual, yet intriguing. For example, one fact is that even though Mars is now a rust-colored desert, it was once like Earth. I find these facts very interesting, so I created a website that is both interesting and engaging for all users, even users without a scientific background.

## How it works
On the website, there are 3 sets of flashcards about the planets, the dwarf planets, and major moons. To go through these sets of flashcards, the user can click the 'Previous' and 'Next' buttons. There is also a small quiz at the bottom that the user can try to see how much they've learned!

## Built with
- HTML
- CSS
- JavaScript
